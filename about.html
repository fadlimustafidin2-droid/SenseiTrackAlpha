SenseiTrack

Sistem informasi penilaian diri untuk guru — membantu mendokumentasikan refleksi mengajar, bukti pembelajaran, dan perkembangan kompetensi guru, kini dengan dukungan multi-akun dan dashboard khusus Admin untuk memantau perkembangan tim pengajar.

Versi 2.0.0

Teknologi yang Digunakan

HTML5 CSS3 JavaScript Tailwind CSS Chart.js Material Symbols Google Fonts — Poppins
